ContextFlow Support

AI Reading Assistant for Chrome

Getting Started

  1. Click the ContextFlow icon in your Chrome toolbar
  2. Go to Settings tab and enter your OpenAI API key
  3. Visit any article or document — ContextFlow will analyze it automatically
  4. Use the heatmap sidebar and progress widget to track your reading

Common Issues

Extension not analyzing pages?

  • Make sure the extension is toggled ON (blue toggle in popup)
  • Check that you've entered a valid OpenAI API key in Settings
  • The page needs at least 1000 characters of text content
  • Some sites are excluded: Gmail, Slack, Discord, Calendar

"Extension context invalidated" error?

  • This happens when the extension updates while a tab is open. Just refresh the page.

Want to clear cached analyses?

  • Open the popup → Settings → scroll down → click "Clear CF Cache"

Getting an API Key

ContextFlow requires an OpenAI API key (free tier). To get one:

  1. Go to platform.openai.com/api-keys
  2. Create an account or sign in
  3. Click "Create new secret key"
  4. Copy the key and paste it into ContextFlow Settings
